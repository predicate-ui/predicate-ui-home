HTML Structure

This template uses the unsemantic 1200 grid system. If you're unfamiliar with the unsemantic 1200 grid system, be sure to read about it here.

All grid units are contained within a top level element, with the class name grid-container. This has a built in "clearfix" so it does not require a clearing element at the end. It also handles centering the grid layout within the page. By default, it has a max width of 1200px.

Here is an example of a simple use case. 

<div class="grid-container">
  <div class="grid-50">
    I am 50% wide.
  </div>
  <div class="grid-25">
    I am 25% wide.
  </div>
  <div class="grid-25">
    I am 25% wide.
  </div>
</div>

There are grid classes named grid-x where "x" is a number that represents the percentage width of each grid unit. These cover multiples of 5, up to 100 — grid-5, grid-10 … grid-95, grid-100. There are also classes for dividing a page into thirds: grid-33 and grid-66 which are 33.3333% and 66.6667% wide, respectively. Each grid-x unit has 10px of padding on the right and left side that, when coupled with the padding from an adjacent grid-x unit (or the grid-container itself), creates a 20px gutter. The reason for this symmetry, versus having 20px spacing on only the left/right is that it allows for push-x and pull-x classes to rearrange grid units, regardless of whether they are at the beginning/end of a row.

Change Site Logo:

By default the template uses a simple text based logo, in order to change it find this code block. 

<div class="site-branding mobile-grid-100 tablet-grid-100">
    <a href="#"><h1>Lansing</h1></a>
</div>

In order to place an image logo you need to exchange the previous declared H1 tag with an image tag like this : 

<div class="site-branding mobile-grid-100 tablet-grid-100">
    <a href="#"><img src="images/logo.png" alt="Your Site Title goes here" /></a>
</div>

Main Navigation:

To add a new menu item you just need to add a new list element. In order to make the navigation work properly you need to set an anchor to the desired section of the page. Learn more about page anchors here

<nav class="mobile-grid-100 tablet-grid-100" id="ut-navigation">
    <ul class="clearfix">
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#gallery">Gallery</a></li>
        <li><a href="#newsletter">Newsletter</a></li>
        <li><a href="#download" class="ut-navigation-cta">Download now<i class="icon-cloud-download"></i></a></li>
    </ul>
</nav>

Change main background header image :

In order to change the main background header image you need to open the style.css and find the paralaxx configuration section around line 430++. The main header image is located inside the .parallax-banner-1 rule declaration.

/* #Parallax
================================================== */
...

.parallax-banner-1 {background-image: url('../images/product-bg.jpg');}
We recommend to use an image size of 1920 x 1080 pixel in order to make the background image fit perfect on all devices. Tip: Just exchange the image by image name, so you do not need to touch the code.

Change newletter background header image :

One code line beneath you can also change the newletter background image.

.parallax-banner-2 {background-image: url('../images/newsletter-bg.jpg');}

Adding an new Product Section:

The template comes with 2 predefined product information sections. One section holds an image left and the description is located to the right and the other section is vice versa. The Basic markup for each section :

Align Product Section Text:

Sometimes it's necessary to change the vertical text align of a text element if the image next to is larger than the text itself. In this case you need to open the style.css and adjust the vertical align inside this code block: 

.product .grid-100 {
	position:relative;
	display:table-cell;
	vertical-align:top;
	padding-top:160px;
	padding-bottom:80px;
}

Adding a new slider image :

For each slider you like to add , you'll need 2 diffierent images. On the one hand the large main image and on the other hand the small thumnbail image which is located inside the small thumnbail navigation beneath the main slider.

<ul class="slides">
    <li data-thumb="images/slide-1.jpg">
        <img src="images/slide-1.jpg" alt="slide-1" />
    </li>
    <li data-thumb="images/slide-2.jpg">
        <img src="images/slide-2.jpg" alt="slide-2" />
    </li>
    <li data-thumb="images/slide-3.jpg">
        <img src="images/slide-3.jpg" alt="slide-3" />
    </li>
    <li data-thumb="images/slide-4.jpg">
        <img src="images/slide-4.jpg" alt="slide-4" />
    </li>
</ul>

Each slide you add needs the following markup. Make sure the data-thumb attribute on the list element refers to the thumbnail image you have created and the img tag to the main large image of this slide.

<li data-thumb="images/slide-4.jpg">
    <img src="images/slide-4.jpg" alt="slide-4" />
</li>
We recommend to use the same image size for all slider images, as well as the same size for all thumnbail images too.

CSS Structure

The Main CSS file for this template is the style.css . As you can see, we are calling this file as the last file inside as list of different files to make sure all style rules are applying correctly. We highly recommend to keep this structure.

<!-- CSS
================================================== -->
<link rel="stylesheet" href="css/reset.css">
<link rel="stylesheet" href="css/ut-responsive-grid.css">
<link rel="stylesheet" href="css/ut-fontface.css">
<link rel="stylesheet" href="css/font-awesome.css">
<link rel="stylesheet" href="css/flexslider.css">
<link rel="stylesheet" href="css/style.css">

JS Structure

Except the main JQUERY library we are calling all necessary script at the end of the HTML Document, before the closing body tag. If you plan to insert a new JQUERY Plugin we recommend to call it inside this section too! If you need to hook the new script, we recommend to place the necessary code inside the custom.js as we already did for parallax and flexslider.

<!-- Load Javascript
================================================== -->
<script src="js/jquery.parallax.min.js"></script>
<script src="js/jquery.flexslider-min.js"></script>
<script src="js/jquery.custom.js"></script>
